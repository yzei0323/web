
장바구니 -세션저장소 사용하기 (저장소)


--세션저장소 : 한 명의 사용자가 계속 유지하면서 사용할 수 있는 저장소
--하나의 브라우저로 접속해서 여러페이지를 요청할 수 있다.  이 때 발생하는 정보를 계속유지하고 싶다면 세션저장소를 사용한다 
-- request 저장소는 응답이 되면 사라진다. 
     

메인페이지 조회하기
상품조회하기  -> 장바구니 담기
장바구니 조회하기  -> 장바구니 삭제
상품상세조회하기  -> 장바구니 담기 




세션은 정해진 시간동안만 유지된다 
여러페이지를 요청하면서 발생하는 데이터를 유지하고 싶다면 세션저장소를 사용한다 
물론 로그아웃경우를 보면 원할 때 세션저장소를 삭제할 수 있다
